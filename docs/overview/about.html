Обзор

VKUI — это библиотека адаптивных React-компонентов, предназначенных для разработки веб-приложений. Основана на одноименной дизайн-системе ВКонтакте ↗.

Преимущества

  • Адаптивность — библиотека подстраивается под различные размеры экранов, обеспечивая удобный интерфейс на смартфонах, планшетах и компьютерах.

  • Мультиплатформенность — интерфейсы VKUI практически идентичны дизайну других платформ, в которые они могут быть интегрированы. Благодаря способности “мимикрировать”, библиотека может адаптироваться под iOS и android клиенты, сохраняя стиль дизайна, характерный для vk.com ↗ и m.vk.com ↗.

  • Разные цветовые схемы — использование различных цветовых схем, включая темную тему, является одним из ключевых аспектов обеспечения доступности.

Поддерживаемые браузеры

Для возможности использования нативной функциональности, каждый год увеличиваются версии поддерживаемых браузеров на основе статистики команды Вконтакте. На данный момент версии следующие:

  • Chromium based 63+ (Chrome, Edge, Chrome for Android и др.);
  • Firefox 55+;
  • Safari 12+ (Safari, iOS).

Релизный цикл

Для версионирования придерживаемся semver ↗. Временные рамки следующие:

  • Патч-версии (x.y.Z) выпускаются по необходимости, при наличии исправлений;
  • Минорные версии (x.Y.z) выпускаются с периодичностью один-два раза в месяц;
  • Мажорные версии (X.y.z) с обратно несовместимыми изменениями выходят один раз в год.

Дизайн

На странице VK в Figma Community ↗ доступны четыре библиотеки:

  • VKUI Common Library [Beta] — основная библиотека компонентов для android, также используется для создания компонентов на десктопе;
  • VKUI iOS Library — библиотеки компонентов для iOS c шаблонами готовых экранов;

Вы можете использовать эти открытые дизайн-библиотеки для реализации своих интерфейсов.

Лицензия

Библиотека VKUI распространяется под лицензией MIT ↗. Вы можете использовать её в любых проектах (в том числе коммерческих), с указанием авторства.
